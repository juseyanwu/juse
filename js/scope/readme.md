-js 是弱类型，动态语言
弱类型指这门语言在声明的时候不需要声明类型
如var 可以声明 var a = 1 也可以声明 var b = 'hello'
js 在执行之前会进行编译，编译可以搞明白你到底声明的是什么类型的变量，在设计之初没有设计为强类型的语言，代码的编译总是会发生在代码执行前一刻
